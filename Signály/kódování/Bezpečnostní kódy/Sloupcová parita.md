## Sloupcová parita
Ochrana před shluky chyb. Průměrná délka shluku 6bitů. [[Parita|Parita]] se počítá pro každý řádek a nakonec i pro sloupce. 

| data   | řádková parita |
| ------ | -------------- |
| 101100 | 1              |
| 111001 | 0              |
| 110001 | 1              |
| 001001 | 0              |
| ...       | ...               |
| 101101 | 0              |

Na posledním řádku je takzvaná sloupcová parita.
