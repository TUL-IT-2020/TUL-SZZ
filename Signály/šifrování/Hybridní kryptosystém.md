# Hybridní kryptosystém
Asymetrická šifra je použita pro přenos jednorázového klíče symetrické šifry. 

Velikosti klíčů:

| Sym. šifry             | Asym. šifry             |
| ---------------------- | ----------------------- |
| 128 b klíč             | 2048 b klíč             |
| 256 b (extra bezpečné) | 4096 b  (extra bezpečné) |
