# Widgets

Dvě kategorie:
- Informační widgety – rychlé zobrazení informací, které se mění v čase (počasí, kalendáře atd.)
- Funkční widgety – zkratky aplikací (zástupci), které umí nějakou činnost ( přehrávače, přidání poznámky, vypnutí WiFi aj.)
