# Eulerova funkce
$\varphi (n)$ ... počet čísel v řadě 1, 2, 3, ..., která jsou nesoudělná s $n$.

$\varphi(p) = p-1$
- $p$ - prvočíslo


$$
\varphi(n) = \prod_{i=1}^{k} p_i^{\alpha}-p_i^{\alpha-1}
$$
Tedy:
$$
\varphi(n) = 
\left( p_1^{\alpha_1}-p_1^{\alpha_1-1} \right) *
... *
\left( p_1^{\alpha_k}-p_k^{\alpha_k-1} \right)
$$
