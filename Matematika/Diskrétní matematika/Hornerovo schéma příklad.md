## Příklad
$$
f(x) = 3x^5 + 2x^3 + 4x + 2
$$
- $\in Z_5[x]$
### Postup
| 3 | 3 | 0 | 2 | 0 | 4 | 2 |
| ---- | ---- | ---- | ---- | ---- | ---- | ---- |
|  | / | $3*3(5)=4$ | 2 | 2 | 1 | 0 |
|  | 3 | $4+0=4$ | 4 | 2 | 0 | 2 |


| 1   | 3   | 0   | 2   | 0   | 4   | 2   |
| --- | --- | --- | --- | --- | --- | --- |
|     | /   | 3    | 3    | 0    | 0    | 4    |
|     | 3    | 3    | 0    | 0    | 4    | 1    |
### Řešení
f(3) = 2
f(1) = 1
