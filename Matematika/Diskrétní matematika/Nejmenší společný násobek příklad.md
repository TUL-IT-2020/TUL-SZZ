# Nejmenší společný násobek 
$$
NSN(a,b) = \frac{a*b}{NSD(a,b)}
$$

## Příklad
NSN(a,b)
a = 13440
b = 19740
### Postup
a = 13440 = 20 * 672
b = 19740 = 20 * 987

| dělenec | dělitel | $q_i$ | zbytek |
| ---- | ---- | ---- | ---- |
| 987 | 672 | 1 | 315 |
| 672 | 315 | 2 | 42 |
| 315 | 42 | 7 | 21 |
| 42 | 21 | 2 | 0 |
NSD(a,b) = 20 * 21 = 420
$$
NSN(13440,b) = \frac{13440*19740}{420} = 631 680
$$
### Řešení
NSN(a,b) = 631 680
