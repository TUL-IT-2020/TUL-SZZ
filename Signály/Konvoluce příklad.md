# Konvoluce
$$
x[x] = \delta[n+1]*x[-1]
$$
## Příklad
$x[n] = 2\delta[n+2] + \delta[n+1] + \delta[n-1] + 3\delta[n-2]$
$h[n] = \delta[n-1] + \delta[n-2] + \delta[n-3]$
### Postup:
Tedy:
- signál `[2,1,0,1,3]`
- impulzní odezva: `[1,1,1]`

| $n$               | -2  | -1  | 0   | 1   | 2   | 3   | 4   | 5   |

| $T(\delta[n-1])$  |     |     |     | 1   | 1   | 1   |     |     |
| $T(3\delta[n-2])$ |     |     |     |     |     | 3   | 3   | 3   |
| $T(x)$            | 0   | 2   | 3   | 3   | 2   | 4   | 4   | 3   |
